# CMIS — 演唱会文创智能策划系统

## 核心理念

> 将艺人的情绪价值、演唱会体验、城市文化与粉丝消费心理，转化为可规模化销售的情绪消费产品系统。

本系统不是「给艺人做周边」，而是一套完整的**商业化情绪产品决策引擎**。

**打造专业文创产品的核心标准：**
1. 产品必须有「只属于这个艺人×这场演唱会×这个场馆」的不可替代性
2. 每款产品都能讲清楚自己的故事（从哪里来→发生了什么→去哪里）
3. 12款产品组成有机矩阵，覆盖不同粉丝层次和消费时间节点
4. 方案在供应链层面可落地执行

## 关键判断原则

### 原则1：情绪优先于功能
粉丝买的是「那个夜晚」，不是一件衣服或一个杯子。产品的情绪价值是第一位的。

### 原则2：审美准确是生死线
审美错误直接导致销量失败。必须从官方物料、粉丝社群、艺人历史视觉体系中交叉验证。

### 原则3：艺人核心符号 > 场馆元素

> ⚠️ 真实销售数据验证的核心原则。

粉丝购买的核心驱动力永远是艺人本身的标志性符号，而非场馆。

**典型案例**：五月天演唱会，最强销售符号是「五个球」，拍拍灯、冰箱贴均围绕五个球。鸟巢元素对销售几乎不构成吸引力。对于知名度没有这么强的艺人，鸟巢元素可以赋能加分。

**场馆元素权重动态判定：**

| 艺人IP强度 | 场馆策略 | 文创表现方式 |
|-----------|---------|------------|
| ≥ 8分（强IP） | 隐性层 | 仅水洗标/经纬度/日期压纹，主视觉100%给艺人核心符号 |
| 6–7分（中IP） | 辅助层 | 场馆元素与艺人符号共存，不抢主视觉 |
| ≤ 5分（弱IP） | 核心层 | 场馆品牌为产品赋能，可作为核心卖点 |

### 原则4：每款SKU必须通过「差异化三问」

每款SKU至少能回答以下一问，三问全无则必须重新设计或删除：

- **Q1 符号特性**：是否有只属于这个艺人/这场演唱会的不可替代元素？
- **Q2 可玩性**：是否有集齐机制/拆解彩蛋/互动玩法/组合逻辑？
- **Q3 传播触发点**：粉丝拍照发小红书的具体理由是什么？

### 原则5：不与艺人现有文创撞车
「品类+价格」双重重叠即为撞车，必须调研现有产品并列出回避清单。

### 原则6：品类从发现中来，不从清单中选

> ⚠️ 这是v3.0的核心升级原则。

不从预设品类库里勾选产品。通过「艺人历史爆款×文创最新趋势×鸟巢场馆特性」三维交叉，主动发现最适合本次演唱会的产品方向。sku-library.md 是发现框架，不是选择清单。

保底品类（票根套/贴纸/T恤/礼盒）确保基本盘，其余品类通过三维框架动态发现。

### 原则7：每款产品有完整叙事

每款产品必须讲清楚：**它从哪里来**（与艺人/演唱会的具体锚点）× **它发生了什么**（粉丝拥有它后会做什么）× **它去哪里**（五年后这个产品对粉丝意味着什么）。

### 原则8：产品矩阵覆盖三类粉丝和四个时间节点

**三类粉丝**：路人粉（入门款）× 核心粉（主力款）× 铁粉（限量稀缺款）
**四个时间节点**：入场前（冲动购买）× 演出中（互动应援）× 演出后现场（情绪余韵）× 演出后线上（开箱传播）

### 原则9：场馆物理适配

鸟巢是夜间户外超大型场馆，产品必须考虑：夜晚可见性、轻便入场性、户外耐用性、大体量传播效率。

### 原则10：定价必须基于真实历史数据
必须搜索艺人历史实际售价，每个价格区间注明数据来源。

### 原则11：方案必须供应链可落地
每款SKU须标注生产周期，与演出日期对比确认时间可行性。

## Layer 1｜数据采集与艺人全景分析

### 目标
构建艺人的**实时全景画像**，重点关注：核心符号资产、历史文创真实数据、粉丝传播行为规律。

### 必须完成的五项调研

#### A. 基础画像
- 艺人简介、出道背景、代表作
- 官方应援色（精确色值）、粉丝昵称
- 粉丝文化特征（圈内梗、应援方式、特殊文化）

#### B. 核心符号资产识别（最重要）

> 这是整个策划的视觉锚点，必须识别清楚。

搜索：`艺人名 应援物料`、`艺人名 粉丝二创`、`艺人名 标志`

**识别框架：**

| 符号类型 | 说明 | 典型例子 | 文创转化方向 |
|---------|-----|---------|------------|
| 专属形状 | 艺人/组合独有的几何或图形符号 | 五月天「五个球」 | 直接成为产品主视觉，可做3D化/发光化 |
| 专属IP形象 | 官方认可的卡通/吉祥物形象 | 汪苏泷「素龙」 | 毛绒/挂件/贴纸主角 |
| 专属应援道具 | 官方或粉丝约定的应援物形态 | 特定形状荧光棒 | 引流款/互动款方向 |
| 专属颜色组合 | 超出单一应援色的组合视觉 | 应援色+演唱会主题色 | 全系列颜色语言 |
| 专属文字/梗 | 粉丝圈内高频的词/口号/梗 | 「大巡特巡」「末日电台」 | 文案/印花文字 |
| 专属气味关联 | 演唱会现场的香氛/特殊体验 | 某演唱会的玫瑰香氛 | 香薰/香氛类产品方向 |

**符号强度判定：**
- 强符号（⭐⭐⭐）：粉丝自发大量使用，无需解释即可识别，只属于这个艺人
- 中符号（⭐⭐）：官方使用，粉丝认知度高，但辨识度有限
- 弱符号（⭐）：仅官方定义，粉丝未内化，不适合作为主文创锚点

#### C. 历史文创产品全面调研（必须搜索）

搜索：`艺人名 演唱会周边`、`艺人名 快闪店 产品`、`艺人名 官方文创 卖断货`

**需要整理的内容：**

| 产品名称 | 品类 | 实际售价 | 销售渠道 | 是否爆款 | 爆款底层原因 |
|---------|-----|---------|---------|---------|-----------|
| [产品] | | ¥[价格] | [渠道/时间] | 是/否 | [符号/可玩/互动/叙事] |

**输出两张清单：**
1. **定价参考表**：各品类历史实际售价，作为本次定价的硬约束
2. **撞车回避清单**：艺人现有在售产品，本次不得重复的品类+理由

#### D. 粉丝社媒传播行为分析（必须搜索）

搜索：`艺人名 演唱会周边晒单`、`艺人名 文创 小红书 微博`

**分析提取：**
- 哪类产品出现在粉丝晒单中（频率）？

### 维度C：鸟巢场馆物理特性适配

| 场馆体量 | 约9万座，四面分布，最远座位距舞台200m+ | 大尺寸/发光/动态产品比精细工艺产品更有现场价值 |
| 户外开放 | 开放式场馆，夏季高温潮湿，有风 | 纸质产品易损；遮阳/防晒类有额外功能价值；轻便产品优于重物 |
| 入场安检 | 严格安检，禁止自带部分物品（如外购荧光棒） | 官方应援物料有天然垄断优势；可穿戴产品入场无障碍 |
| 人流密度 | 单场次近9万人，入/退场拥挤 | 可穿戴产品（T恤/帽衫/帽子）在人流中形成大面积视觉传播 |
| 观演距离 | 大部分观众距舞台较远 | 望远镜类产品有实用需求；大屏互动比精细产品更有现场感 |
| 周边配套 | 鸟巢周边商业区、地铁站人流 | 销售前移到场馆外区域有效；轻便易携带产品更适合 |

**鸟巢场馆加分产品特征：**
- 夜晚可发光/反光（LED/镭射/荧光涂层）
- 大面积可视（挥舞时在远处也能看到）
- 轻便易携（不影响观演体验）
- 可穿戴（T恤/帽子/腰包，入场即成活广告）
- 防潮防风（夏季户外实用性）

**鸟巢场馆减分产品特征：**
- 精细工艺但夜晚无法欣赏
- 体积大/重量重（入场不便，观演期间无处放置）
- 纸质易损（夏季汗水/雨水风险）
- 仅适合近距离观看的小巧精致品（在9万人场馆中传播效率极低）

## 二、产品叙事完整性要求

> ⚠️ 好的文创产品都有自己的故事。粉丝买的不只是物品，是「那个夜晚」的一个具体切片。

每款SKU在确定前，必须能讲清楚以下叙事：

**「这个产品的故事」三要素：**

1. **它从哪里来**：与艺人/演唱会/本次主题的具体连接是什么？（不是泛泛的「纪念品」，而是有具体锚点）
2. **它发生了什么**：粉丝拥有它之后会发生什么？在现场用它？回家摆放它？送给朋友它？集齐系列？
3. **它去哪里**：五年后，这个产品对拥有它的人意味着什么？会被保留吗？为什么？

> 能讲清楚这三个问题的产品，才有持久的情绪价值。
> 讲不清楚的，是功能性商品，不是文创产品。

## 三、产品组合整体协调性检查

> ⚠️ 12款产品不是12个独立产品的堆砌，而是一个有机的产品矩阵。

**最终方案输出前，必须完成以下整体性检查：**

### 3.1 价格梯度自然引导

```
引流款（¥XX）→ 主力款（¥XX）→ 高端款（¥XX）→ 稀缺款（¥XX）
```
检查：相邻价格区间的跨度是否合理？是否存在断层（某个区间没有产品，导致消费者直接跳级）？

### 3.2 视觉整体感

- 全系列产品放在一起，是否有统一的视觉语言？
- 核心符号是否在不同价位产品中以不同形式出现（引流款简版→稀缺款精装版）？
- 颜色体系是否统一，不同产品之间是否「认识彼此」？

### 3.3 粉丝分层覆盖

| 粉丝类型 | 特征 | 本次方案中对应的产品 |
|---------|-----|-----------------|
| 路人粉/新粉 | 只来一场，预算有限，想要「到此一游」的感知 | ¥35-79 引流款，设计要有入门门槛感 |
| 核心粉 | 多场次，有收藏意愿，愿意为「只有我有」付费 | ¥199-399 主力/品牌款 |
| 铁粉/追随者 | 外地专程赶来，把这场当人生大事，要「最好的那个」 | 限量礼盒/稀缺款，叙事要有史诗感 |

检查：三类粉丝是否都有对应的产品？有没有哪类粉丝被忽略？

### 3.4 现场动线覆盖

| 时间节点 | 消费状态 | 对应产品 |
|---------|---------|---------|
| 入场前（排队等候） | 兴奋期待，冲动消费高峰 | 引流款，快速决策，低价 |
| 演出中（高潮段落） | 情绪峰值，愿意为「这个瞬间」付费 | 应援互动类产品 |
| 演出结束后（出场） | 情绪余韵，不舍，「我想带走这个夜晚」 | 稀缺款/礼盒，高情绪密度 |
| 演出后线上 | 沉淀回味，晒单分享 | 开箱类产品，延续传播 |

检查：每个时间节点是否有对应产品？有没有断层？

## 五、保底品类清单（确保基本盘不遗漏）

> 以下品类是演唱会场景下几乎零失败率的「安全选项」，
> 即使创新不足，也应作为方案的基础配置存在。
> **但这些品类的设计必须基于艺人核心符号，而非通用设计。**

| 品类 | 原因 | 设计底线要求 |
|-----|-----|------------|
| 票根保护套 | 买票即刚需，几乎100%转化率 | 必须融入本次演唱会主视觉，内侧有叙事细节 |
| 贴纸套组 | 最低门槛引流，万能附加品 | 图案必须基于艺人核心符号，不可用通用图案 |
| T恤（1-2款） | 最高利润贡献，演唱会标配 | 版型+设计都要过关；演出前必须粉丝内测 |
| 限量礼盒 | 铁粉必购，客单价天花板 | 必须有「情绪彩蛋」（信件/专属卡/特殊内含物） |

## 六、品类禁入清单（已验证的失败方向）

> 以下方向在演唱会文创场景中有较高失败概率，选用前需有充分的差异化理由。

| 禁入方向 | 失败原因 | 例外情况 |
|---------|---------|---------|
| 通用设计产品（任何艺人都能用的） | 粉丝感知「不是专属的」，购买动机弱 | 无例外 |
| 与艺人现有文创重叠的品类 | 粉丝感知「官方店有卖，没有新意」 | 无例外 |
| 低质感塑料制品 | 与演唱会的情绪价值严重不匹配 | 无例外 |
| 定价虚高但内含物不足的礼盒 | 粉丝感知「不值」，口碑负面 | 需确保内含物感知价值超过定价30%+ |
| 功能性强但设计平庸的日用品 | 如「艺人头像+普通保温杯」，是功能品不是文创 | 需有极强的设计/符号支撑 |
| 三问全无的大路货 | 没有符号特性/可玩性/传播触发点 | 无例外 |
| 不适合鸟巢物理条件的产品 | 如精细工艺在夜晚无法欣赏，大件入场困难 | 需评估现场实际体验 |

## 七、「新意品类」发现思路（开放性指引）

> 以下是帮助打破常规、发现新品类的思考路径，不是列表，是问题。

**从艺人专属元素出发：**
- 艺人有没有专属的「形状」可以产品化？（五月天的球、某艺人的手势）
- 艺人有没有专属的「气味」关联？（香氛/香薰蜡烛是近年文创增速最快的品类之一）
- 艺人有没有专属的「声音」可以实体化？（音波图/声纹周边）
- 艺人有没有专属的「故事场景」可以空间化？（场景复刻摆件/微缩模型）
- 艺人有没有专属的「颜色」足够独特，可以做成色卡/调色板文创？

**从粉丝行为出发：**
- 粉丝在演唱会现场最典型的「动作」是什么？能不能为这个动作设计专属产品？
- 粉丝演唱会后最想「留住」的是什么瞬间？能不能用产品承载？
- 粉丝最常把演唱会周边「放在哪里/用在哪里」？能不能设计一个让这个场景更完整的产品？

**从鸟巢场馆出发：**
- 鸟巢的夜晚有什么可以产品化的视觉元素？（钢格纹/灯光/星空）
- 鸟巢的超大规模有什么可以玩的数字概念？（「第X号见证者」「坐标区X排X号」）
- 鸟巢的户外属性有什么实用需求没被满足？（防晒/降温/遮光有没有可以做成文创的方式？）

**从文创市场跨界出发：**
- 博物馆/美术馆最近爆了什么新品类？能不能迁移到演唱会场景？
- 潮玩/盲盒领域有什么新玩法？能不能引入？
- 快闪店/快消品有什么跨界联名形式？能不能借鉴？

## 八、历史爆款底层逻辑总结

> 这是比「哪些产品卖得好」更有价值的知识：**为什么**这些产品卖得好。

| 底层逻辑 | 说明 | 典型产品形态 |
|---------|-----|------------|
| 符号垄断 | 只有这个艺人才有这个符号，拥有它就是身份标签 | 专属形状/IP形象/独特颜色组合 |
| 集齐驱动 | 有多款，粉丝想集齐，单次购买变多次购买 | 分枚纪念章/分款贴纸/分场次限定 |
| 现场仪式 | 在演唱会现场使用，与「那个夜晚」绑定 | 应援棒/可挥舞产品/可点亮产品 |
| 叙事彩蛋 | 产品里藏着只有粉丝懂的故事/梗/情感 | 礼盒内信件/内衬隐藏文字/特殊包装 |
| 可晒性 | 天然适合拍照发社媒，产品即传播载体 | 开箱体验/穿戴上身即传播/夜晚发光效果 |
| 唯一性证明 | 编号/限量，证明「我是那X个人之一」 | 带编号的限量产品 |
| 实用延续 | 演唱会后日常仍在使用，每次使用=持续回忆 | 可穿戴/马克杯/香薰 |
