
# CMIS — 品类发现框架与产品决策指南 v4.0

> ## 核心使用原则
>
> **本文件不是「品类选择清单」，而是「品类发现方法论」。**
>
> 错误用法：从下面的列表里勾选产品组成方案。
> 正确用法：用下面的框架和问题，主动发现「只适合这个艺人、
> 这场演唱会、这个场馆」的产品。
>
> 文创市场品类极其丰富，远超任何列表所能覆盖。
> 真正的爆款往往出现在通用清单之外——五月天的发光球、
> 汪苏泷的素龙挂件、各艺人定制香氛蜡烛……
> 这些产品的共同点：都是从「这个艺人+这个时刻」出发推导出来的，
> 而不是从货架上挑的。
>
> **模式B/C额外原则**：
> 经纪公司指定品类时，本文件用于「在指定品类内找到最优创意实现方式」，
> 以及「识别指定框架中可能存在的产品机会盲点」。

## 一、品类发现三维框架（核心方法）

每次策划必须经过三个维度的交叉搜索：

```
        艺人历史爆款
           （已验证的粉丝偏好与底层逻辑）
               ↕
    文创市场最新趋势 ←→ 场馆物理特性
       （避免俗套）      （鸟巢专项）
```

### 维度A：艺人历史爆款底层逻辑提取

**通过 web search 完成，不可依赖训练数据。**

搜索目标：
- `艺人名 演唱会周边 爆款`
- `艺人名 快闪店 产品`
- `艺人名 官方周边 卖断货`
- `艺人名 粉丝晒单 小红书`

**提取逻辑（不照搬产品，提取底层原因）：**

| 历史爆款产品 | 爆火原因分析 | 底层逻辑 | 本次可延伸的方向 |
|------------|-----------|---------|--------------|
| [产品名] | [为什么卖爆] | [可玩/符号/互动/叙事] | [如何应用这个逻辑] |

> 核心问题：粉丝买了之后做了什么？
> 用了它？拍了它？集了它？给别人看了它？
> 那个「做了什么」才是真正的爆款原因。

### 维度B：文创市场最新趋势扫描

**每次策划前必须实时搜索，不可使用过期认知。**

搜索目标：
- `演唱会文创 新品类 2025 2026`
- `IP周边 新玩法 潮流`
- `博物馆文创 爆款 最新`
- `快闪店 周边 新品`
- `谷子 新品类 2026`

**评估框架：**

| 新兴品类/玩法 | 市场热度 | 艺人适配度 | 鸟巢场馆适配度 | 是否推荐引入 |
|------------|---------|---------|------------|-----------|
| [品类] | 高/中/低 | 强/中/弱（原因） | 强/中/弱（原因） | 是/否 |

> 重点关注演唱会圈子之外的文创创新（博物馆/快闪/潮玩），
> 往往比演唱会圈内趋势更超前，更容易带来差异化。

### 维度C：鸟巢场馆物理特性适配

> ⚠️ 这个维度在所有通用框架中都缺失，是本系统对鸟巢专项策划的独有模块。

**鸟巢物理特性清单（每次策划必须对照）：**

| 特性 | 具体参数/条件 | 对产品的影响 |
|-----|------------|-----------|
| 演出时段 | 夜间（19:00后，夏季约20:00后天黑） | 发光/反光类产品视觉效果极强；哑色产品夜晚辨识度低 |
| 场馆体量 | 约9万座，最远距舞台200m+ | 大尺寸/发光/动态产品比精细工艺产品更有现场价值 |
| 户外开放 | 开放式，夏季高温潮湿，有风 | 纸质产品易损；轻便产品优于重物；遮阳有额外功能价值 |
| 入场安检 | 严格，禁止部分自带物品 | 官方应援物料有天然垄断优势；可穿戴产品入场无障碍 |
| 人流密度 | 单场近9万人，入/退场拥挤 | 可穿戴产品在人流中形成大面积视觉传播 |
| 观演距离 | 大部分观众距舞台较远 | 大视觉产品优于精致小物；发光类传播效率更高 |
| 周边配套 | 鸟巢周边商业区、地铁站人流 | 销售前移到场馆外区域有效；轻便易携带产品更适合 |

**鸟巢加分产品特征：**
- 夜晚可发光/反光（LED/镭射/荧光涂层）
- 大面积可视（挥舞时远处也能看到）
- 轻便易携（不影响观演体验）
- 可穿戴（T恤/帽子，入场即成活广告）
- 防潮防风（夏季户外实用性）

**鸟巢减分产品特征：**
- 精细工艺但夜晚无法欣赏
- 体积大/重量重（入场不便）
- 纸质易损（夏季汗水/雨水风险）
- 仅适合近距离观看的小巧精致品

## 二、产品叙事完整性要求

> 好的文创产品都有自己的故事。
> 粉丝买的不只是物品，是「那个夜晚」的一个具体切片。

每款SKU确定前，必须能讲清楚叙事三要素：

**1. 它从哪里来**
与艺人/演唱会/本次主题的具体连接是什么？
（不是泛泛的「纪念品」，而是有具体锚点）

**2. 它发生了什么**
粉丝拥有它之后会发生什么？
在现场用它？回家摆放？送给朋友？集齐系列？

**3. 它去哪里**
五年后，这个产品对拥有它的人意味着什么？会被保留吗？为什么？

> 能讲清楚三要素的产品，才有持久的情绪价值。
> 讲不清楚的，是功能性商品，不是文创产品。

> 模式C：指定品类同样须完成叙事三要素构建。
> 如指定品类天然叙事薄弱，在约束项审查中提供「叙事强化建议」。

## 三、产品组合整体协调性检查

### 3.1 价格梯度自然引导

```
引流款（¥XX）→ 主力款（¥XX）→ 高端款（¥XX）→ 稀缺款（¥XX）
```
检查相邻价格区间跨度是否合理，是否存在购买意愿断层。

### 3.2 视觉整体感

- 全系列产品放在一起，是否有统一的视觉语言？
- 核心符号是否在不同价位产品中以不同形式出现？
- 颜色体系是否统一？

> 模式C：视觉整体感检查须包含：
> 「经纪公司指定的视觉元素与全系列的视觉一致性如何？是否有突兀感？」

### 3.3 粉丝分层覆盖

| 粉丝类型 | 消费特征 | 对应产品 | 是否覆盖完整 |
|---------|---------|---------|-----------|
| 路人粉/新粉 | 只来一场，预算有限，「到此一游」感知 | | |
| 核心粉 | 多场次，有收藏意愿，要「只有我有」 | | |
| 铁粉/追随者 | 外地赶来，把这场当人生大事 | | |

### 3.4 现场动线覆盖

| 时间节点 | 消费状态 | 对应产品 |
|---------|---------|---------|
| 入场前（排队等候） | 冲动消费高峰 | |
| 演出中（高潮段落） | 情绪峰值 | |
| 演出结束后（出场） | 情绪余韵，不舍 | |
| 演出后线上 | 沉淀回味，晒单 | |

**主要品类生产周期参考：**

| 品类 | 最短周期 | 标准周期 | 注意事项 |
|-----|---------|---------|---------|
| 贴纸/透卡 | 7天 | 10-15天 | 镭射/特殊工艺+5天 |
| 亚克力挂件 | 7天 | 10-15天 | 多色/渐变+3天 |
| T恤（普通印花） | 10天 | 15-20天 | 特殊工艺+7天；多尺码+3天 |
| 帽衫/卫衣 | 15天 | 20-30天 | 定制颜色+7天 |
| 帽子（刺绣款） | 15天 | 20-25天 | |
| 毛绒公仔/挂件 | 25天 | 30-45天 | 开新模具+15-20天 |
| 金属徽章/纪念章 | 30天 | 45-60天 | 浮雕+10天；开模+15天 |
| 精装礼盒（外盒） | 20天 | 28-35天 | 烫金/特殊工艺+7天 |
| 歌词手账/精装书 | 20天 | 25-35天 | 特种纸+5天 |
| 黑胶装饰盘 | 20天 | 30-40天 | |
| 搪瓷杯/马克杯 | 20天 | 30-40天 | |
| 香氛/蜡烛 | 15天 | 20-30天 | 定制香型+10天 |
| 发光/LED产品 | 20天 | 30-45天 | 电子元件+测试时间 |

**演出前关键节点倒推：**

```
演出日期 - 60天：锁定所有供应商，完成打样确认
演出日期 - 45天：长周期产品（金属章/毛绒/礼盒）开始批量生产
演出日期 - 30天：所有产品进入生产
演出日期 - 15天：预售开启，收集真实需求数据
演出日期 -  7天：全部产品到货，完成质检
演出日期 -  3天：现场陈列搭建完成
```

> ⚠️ 如方案中的SKU在现有时间节点内无法完成生产，
> 必须标注风险并提供替代方案或调整建议。

## 五、保底品类清单

> 以下品类是演唱会场景下几乎零失败率的基础配置。
> **但设计必须基于艺人核心符号，而非通用设计。**

| 品类 | 原因 | 设计底线要求 |
|-----|-----|------------|
| 票根保护套 | 买票即刚需，接近100%转化率 | 融入本次演唱会主视觉，内侧有叙事细节 |
| 贴纸套组 | 最低门槛引流，万能附加品 | 图案必须基于艺人核心符号，不可用通用图案 |
| T恤（1-2款） | 最高利润贡献，演唱会标配 | 版型+设计都要过关；演出前粉丝内测 |
| 限量礼盒 | 铁粉必购，客单价天花板 | 必须有「情绪彩蛋」（信件/专属卡/特殊内含物） |

## 六、品类禁入清单

| 禁入方向 | 失败原因 | 例外 |
|---------|---------|-----|
| 通用设计产品（任何艺人都能用） | 「不是专属的」感知，购买动机弱 | 无例外 |
| 与艺人现有文创重叠的品类 | 「官方店有，没新意」感知 | 无例外 |
| 低质感塑料制品 | 与演唱会情绪价值严重不匹配 | 无例外 |
| 叙事三要素全部缺失的产品 | 是功能品不是文创品 | 无例外 |
| 差异化三问全部无答案的产品 | 大路货，粉丝不晒不传播 | 无例外 |
| 不适合鸟巢物理条件的产品 | 体验差，传播效率低 | 须评估实际体验后决定 |
| 定价虚高但感知价值不足 | 口碑负面，损害品牌 | 须确保感知价值超定价30%+ |

## 七、「新意品类」发现思路

> 以下是帮助打破常规的思考路径，不是列表，是问题。
> 每次策划时快速过一遍，往往能发现意料之外的好产品。

**从艺人专属元素出发：**
- 艺人有没有专属的「形状」可以产品化？
- 艺人有没有专属的「气味」关联？（香氛/香薰蜡烛是近年增速最快品类之一）
- 艺人有没有专属的「声音」可以实体化？（音波图/声纹周边）
- 艺人有没有专属的「故事场景」可以空间化？（场景复刻摆件/微缩模型）
- 艺人有没有专属的「颜色」足够独特，可以做色卡/调色盘文创？

**从粉丝行为出发：**
- 粉丝在演唱会现场最典型的「动作」是什么？能不能为这个动作设计专属产品？
- 粉丝演唱会后最想「留住」的是什么瞬间？能不能用产品承载？
- 粉丝最常把演唱会周边「放在哪里/用在哪里」？能不能设计让这个场景更完整的产品？

**从鸟巢场馆出发：**
- 鸟巢的夜晚有什么可以产品化的视觉元素？（钢格纹/灯光/星空）
- 鸟巢的超大规模有什么可以玩的数字概念？（「第X号见证者」「区X排X号」）
- 鸟巢的户外属性有什么实用需求没被满足？（防晒/降温/遮光能否文创化）

**从文创市场跨界出发：**
- 博物馆/美术馆最近爆了什么新品类？能不能迁移到演唱会场景？
- 潮玩/盲盒领域有什么新玩法？能不能引入？
- 快闪店/快消品有什么跨界联名形式值得借鉴？

## 八、历史爆款底层逻辑总结

| 底层逻辑 | 说明 | 典型产品形态 |
|---------|-----|------------|
| 符号垄断 | 只有这个艺人才有这个符号，拥有它就是身份标签 | 专属形状/IP形象/独特颜色组合 |
| 集齐驱动 | 有多款，粉丝想集齐，单次变多次购买 | 分枚纪念章/分款贴纸/分场次限定 |
| 现场仪式 | 在演唱会现场使用，与「那个夜晚」绑定 | 应援棒/可挥舞产品/可点亮产品 |
| 叙事彩蛋 | 产品里藏着只有粉丝懂的故事/梗/情感 | 礼盒内信件/内衬隐藏文字 |
| 可晒性 | 天然适合拍照发社媒，产品即传播载体 | 开箱体验/穿戴上身/夜晚发光效果 |
| 唯一性证明 | 编号/限量，证明「我是那X个人之一」 | 带编号的限量产品 |
| 实用延续 | 演唱会后日常仍在使用，每次使用=持续回忆 | 可穿戴/马克杯/香薰 |
| 同款共鸣 | 与演唱会现场某个视觉/道具同款，穿戴即融入 | 与彩带/荧光棒同色应援帽衫 |
