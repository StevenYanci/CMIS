# CMIS — 6层分析执行规范

## Layer 1｜数据采集与艺人分析

### 目标
通过 web search 构建艺人的**实时全景画像**，重点关注"当下情绪"和"核心符号资产"，而非历史标签。

### 必须检索的内容

#### A. 基础资料
- 艺人/组合简介、出道背景
- 代表作品（歌曲/专辑/影视）
- 历史演唱会规模与口碑
- 官方应援色、粉丝昵称
- 粉丝文化特征（应援方式、圈内梗、圈内文化）

#### B. 核心符号资产（新增，必查）

> ⚠️ 这是比应援色更重要的分析项。粉丝反复使用的符号才是文创的真正锚点。

搜索关键词：`艺人名 应援 二创`、`艺人名 粉丝自制周边`、`艺人名 演唱会应援物料`

**需要识别的核心符号类型：**

| 符号类型 | 举例 | 文创转化方向 |
|---------|------|------------|
| 专属图形/形状 | 五月天「五个球」、某艺人专属手势 | 直接成为产品主视觉 |
| 专属IP形象 | 汪苏泷「素龙」、艺人Q版卡通形象 | 毛绒/挂件/贴纸主角 |
| 专属色彩组合 | 应援色、官方演唱会主色 | 全系列产品必须包含 |
| 专属文字/梗 | 粉丝圈内高频使用的词语/口号 | 文案/贴纸/印花文字 |
| 专属道具 | 官方应援棒形态、特殊形状荧光棒 | 引流款/互动款设计方向 |

**核心符号强度判定：**
- 强符号：粉丝自发大量在二创/应援中使用，无需解释即可识别
- 弱符号：仅官方使用，粉丝认知度低，不适合作为主要文创锚点

#### C. 艺人现有文创产品调研（新增，必查）

> ⚠️ 必须在提出任何SKU建议前完成此调研，防止与艺人现有产品撞车。

搜索关键词：`艺人名 官方周边`、`艺人名 文创`、`艺人名 官方旗舰店`、`艺人名 演唱会周边`

**需要盘点的内容：**

| 维度 | 内容 |
|------|------|
| 在售品类 | 当前官方在售的所有周边品类（T恤/公仔/挂件/手册等） |
| 价格体系 | 各品类实际售价区间（这是定价的硬约束依据） |
| 视觉风格 | 现有产品的设计语言、主色调、图形风格 |
| 热销/滞销信号 | 哪些产品已售罄/长期有货，反映真实销售状况 |
| 撞车品类清单 | 列出本次策划必须回避的品类（品类+价格双重重叠则撞车） |

#### D. 粉丝社媒文创传播分析（新增，必查）

> ⚠️ 粉丝愿意晒出的产品，才是真正打动他们的产品。

搜索关键词：`艺人名 演唱会周边晒单`、`艺人名 文创 小红书`

分析：哪类产品出现在粉丝晒单中？为什么他们愿意拍照分享？提取「传播触发点」。

#### E. 实时热点

搜索关键词：`艺人名 最近 热搜`、`艺人名 小红书 2025/2026`

| 项目 | 内容 |
|------|------|
| 最近热搜话题 | |
| 最近爆火内容 | |
| 粉丝当前情绪方向 | |
| 是否有争议或特殊事件 | |

#### F. 粉丝审美分析

| 审美类型 | 权重 | 代表符号 |
|---------|------|---------|
| 高级极简 | | 黑白灰、无衬线字体 |
| 可爱风 | | 圆润、马卡龙色 |
| 潮流风 | | 街头、解构、logo |
| 文艺风 | | 胶片感、文字、复古 |
| 二次元风 | | 动漫美学、夸张色彩 |
| 复古风 | | 旧纸质、黑胶、磁带 |
| 朋克/暗黑风 | | 金属、对比色 |

## Layer 2｜演唱会情绪系统分析

### 目标
将演唱会解码为**情绪产品的设计蓝图**，并构建专属于本次演唱会的情绪叙事。

### 执行步骤

#### A. 主题解析
搜索演唱会官宣信息、主视觉海报、官方宣传文案。

提取：
- 主主题（科幻/怀旧/情绪/嘉年华/史诗等）
- 官方核心概念文案（原文）
- 情绪关键词（3-5个）
- 世界观设定与叙事方向

#### B. 视觉元素提取
从官宣物料中提取：
- 主色调（精确到色值范围）
- 辅色
- 主视觉元素（图形/符号/材质感）
- 字体风格
- 整体调性（冷/暖/中性）

注意：提取结果必须与 Layer 1 识别的「核心符号资产」对比，确认是否延续或更新。

#### C. 情绪曲线构建

> ⚠️ 情绪曲线应根据艺人特点和本次主题个性化构建，而非套用固定五段模板。

根据艺人演唱会历史规律与本次主题，构建专属情绪时间轴：

| 阶段 | 时长估算 | 情绪状态 | 代表曲目（如有） | 适配SKU方向 | 可能的传播时机 |
|------|---------|---------|----------------|------------|-------------|
| [阶段名] | | [情绪描述] | | | |

说明：「传播时机」指本阶段哪类产品最可能被粉丝拍照传播（如：挥舞类产品在高潮段传播率最高）。

### Layer 2 输出小结格式

```
## 演唱会情绪地图

**主题**：[主题名称]
**官方核心文案**：[原文]
**核心情绪**：[情绪1] + [情绪2] + [情绪3]
**主色调**：[颜色描述，附HEX参考]
**视觉核心元素**：[元素列表]
**本次主题与艺人长期符号的关系**：[延续/更新/新增]

**专属情绪曲线**：
[根据本次演唱会特点个性化描述，不套用通用五段模板]

**情绪×SKU绑定逻辑**：
- [情绪段落] → 推荐 [SKU类型] → 传播触发场景：[描述]
```

## Layer 5｜商品决策与爆款预测

> 综合前4层所有分析结果，生成SKU矩阵。**必须通过差异化三问检查和撞车检查。**

### 前置：差异化三问强制检查

在生成任何SKU之前，先完成以下强制思考：

**Q：本次演唱会最强的「三类产品角色」分别是什么？**

| 角色 | 定义 | 本次对应方向 |
|------|------|------------|
| 现场互动神器 | 演出过程中能被大量使用、形成视觉效果的产品 | [回答] |
| 拍照传播触发器 | 粉丝最想拍照发到社媒的产品（开箱/挥舞/穿戴） | [回答] |
| 核心符号载体 | 最直接承载艺人标志性符号的产品 | [回答] |

每种角色至少对应一款SKU，且三款产品的设计方向必须有所不同。

### SKU四象限结构

每个象限的SKU数量建议：

| 象限 | 定位 | 数量 | 价格范围（参考历史数据校准） |
|------|------|------|--------------------------|
| **引流款** | 低价高传播，冲动消费 | 2-3个 | 根据艺人历史数据确定 |
| **利润款** | 高频使用，情绪价值强 | 3-4个 | 根据艺人历史数据确定 |
| **品牌款** | 提升调性，社交传播 | 1-2个 | 根据艺人历史数据确定 |
| **稀缺款** | 高收藏价值，限量 | 1-2个 | 根据艺人历史数据确定 |

> ⚠️ 价格范围不套用通用区间，必须根据 Layer 1 调研的艺人历史实际售价确定。

### SKU表格模板

每个SKU必须包含：

| 字段 | 内容 |
|------|------|
| SKU名称 | |
| 所属象限 | 引流/利润/品牌/稀缺 |
| 产品描述 | 2-3句话 |
| 核心符号锚点 | 使用了艺人哪个核心符号资产？ |
| 场馆元素层级 | 隐性/辅助/主视觉/无（必须与Layer 3结论一致） |
| **差异化三问回答** | Q1符号特性：[回答] / Q2可玩性：[回答] / Q3传播点：[回答] |
| **撞车检查** | 是否与艺人现有文创重叠？[是/否，说明] |
| 设计方向 | 简述视觉概念 |
| 建议零售价 | [价格，注明定价依据] |
| 建议首批数量 | |

> ⚠️ 如果「差异化三问」三问全部回答为「无」，该SKU不得进入方案，必须重新设计。
> ⚠️ 如果「撞车检查」结论为「是」，该SKU不得进入方案，必须改变品类或设计方向。

### 爆款概率评分模型（升级版）

每个SKU按以下维度评分（总分40分）：

| 维度 | 满分 | 说明 |
|------|-----|------|
| 情绪强度 | 8分 | 与粉丝当前情绪热点契合度 |
| 符号独特性 | 8分 | 是否使用了艺人不可替代的核心符号（差异化三问Q1） |
| 现场互动性 | 6分 | 演出过程中能否被使用/挥舞/佩戴，能否形成视觉效果 |
| 传播触发性 | 6分 | 是否有明确的拍照/晒单/传播场景（差异化三问Q3） |
| 可玩性 | 4分 | 是否有集齐/拆解/组合玩法（差异化三问Q2） |
| 粉丝契合度 | 4分 | 与粉丝审美偏好的匹配度 |
| 收藏价值 | 2分 | 未来是否会保留/升值 |
| 城市适配度 | 2分 | 对本城市消费者的吸引力 |

**爆款概率等级**：
- 总分 ≥ 32分 → ⭐⭐⭐ 强烈推荐主打款
- 总分 24-31分 → ⭐⭐ 稳健款
- 总分 16-23分 → ⭐ 补充款
- 总分 < 16分 → 建议放弃或重新设计
