# CMIS — 7层分析执行规范

## Layer 1｜数据采集与艺人分析（含专属符号词库构建）

### 目标
通过 web search 构建艺人的**实时全景画像**，并提炼出结构化的「艺人专属元素词库」与「禁用元素黑名单」，作为后续所有SKU设计的强制约束基准。

### 必须检索的内容

#### A. 基础资料
- 艺人/组合简介、出道背景
- 代表作品（歌曲/专辑/影视）
- 历史演唱会规模与口碑
- 官方应援色、粉丝昵称
- 粉丝文化特征（应援方式、圈内文化）

#### B. 近3年巡演视觉沉淀（新增，经纪公司第一审核权重）
搜索关键词参考：`艺人名 巡演 视觉`、`艺人名 演唱会 舞台`、`艺人名 演唱会 主视觉`

重点提取：
| 项目 | 内容 |
|------|------|
| 固定主色调与色系变化 | |
| 巡演LOGO/字体风格 | |
| 舞美结构特征（装置/光墙/裸眼3D等） | |
| 经典舞台造型与固定出场姿势 | |
| 视觉调性关键词（冷/暖/科幻/怀旧/极简等） | |

> ⚠️ 这3年的积累是"艺人官方视觉基因库"，本场限定款的设计必须与之调性一致，不能断层。

#### C. 实时热点
搜索关键词参考：`艺人名 最近 热搜`、`艺人名 小红书`、`艺人名 粉丝`

重点提取：
| 项目 | 内容 |
|------|------|
| 最近热搜话题 | |
| 最近爆火内容 | |
| 粉丝当前情绪方向 | |
| 社交高传播内容 | |
| 是否有争议或特殊事件 | |

#### D. 音乐内容沉淀（设计素材来源）
- 高频歌词意象（出现频率最高的意象词汇）
- 核心专辑概念与视觉
- 贯穿多年的创作主题、个人人生态度
- 经典金句、口头禅、演唱会口号

#### E. 独有标志性符号（最高优先级设计素材）
- 专属手势
- 固定舞台道具（如：麦架、特殊乐器、标志性演出服元素）
- 个人LOGO或艺名变体字
- 应援符号（荧光棒颜色/形状/粉丝应援手势）
- 经典动作剪影

#### F. 粉丝审美分析
判断粉丝群体的主流审美偏向（可多选，需标注权重）：

| 审美类型 | 权重 | 当下粉丝偏好信号 |
|---------|------|----------------|
| 高级极简 | | 低饱和、无衬线字体、留白 |
| 暗纹/浮雕质感 | | 肌理感、触觉体验、隐形元素 |
| 文艺复古 | | 胶片感、旧纸质、黑胶风 |
| 潮流街头 | | 解构、混搭、oversize |
| 可爱治愈 | | 圆润、柔和色、毛绒质感 |
| 二次元 | | 动漫美学、夸张色彩 |

> ⚠️ 审美判断错误是文创失败第一大原因；需综合微博超话、小红书、粉丝群高频词交叉验证。排斥廉价卡通风和夸张大LOGO是当下多数粉丝群的共识。

## Layer 1.5｜艺人自有IP生态与分流率分析（核心新增层）

> 📌 此层为 CMIS 关键差异化能力，必须执行，不可跳过。
> 详细方法论与案例库见 `references/artist-ip-ecosystem.md`，执行本层前应先读取该文件。

### 目标

回答核心问题：**这位艺人的粉丝，除了演唱会限定渠道之外，还有多少其他长期、稳定、低门槛的方式可以获得与其相关的产品？** 这直接决定演唱会限定款的"边际稀缺性"和最终产量基数。

### 必须检索的内容（web search）

| 检索方向 | 搜索关键词参考 |
|---------|---------------|
| 自有品牌 | `艺人名 品牌`、`艺人名 创业`、`艺人名 代言 长期` |
| 常规文创产品线 | `艺人名 官方店铺`、`艺人名 周边 官网`、`艺人名 文创` |
| 直播带货/电商 | `艺人名 直播带货`、`艺人名 抖音 直播`、`艺人名 淘宝店` |
| 往届演唱会限定款销售反馈 | `艺人名 演唱会 周边 测评`、`艺人名 演唱会 周边 销量`、`艺人名 演唱会 限定 好评/差评` |

### 执行步骤

#### Step 1：渠道盘点
按四个分流来源逐一排查（参考 `artist-ip-ecosystem.md` 中的分类A-D）：
- A. 自有品牌/跨界品牌（如餐饮、潮牌等与个人IP高度绑定的商业实体）
- B. 长期常规文创产品线（官方店铺常驻款）
- C. 直播带货/常态化电商渠道
- D. 历史限定款"通货化"程度（历年设计是否高度相似）

#### Step 2：四维度评分（每项1-5分）

参照评分表，对每个维度打分，求和得到综合评分（4-20分）。

#### Step 3：分流率分级与产量调整系数

| 总分区间 | 分流率等级 | 产量调整系数 |
|---------|-----------|------------|
| 4-7分 | 低分流 | ×1.0 |
| 8-12分 | 中分流 | ×0.8-0.9 |
| 13-16分 | 高分流 | ×0.6-0.75 |
| 17-20分 | 极高分流 | ×0.4-0.6 |

> ⚠️ 此系数将在 Layer 5 中应用到"建议首批数量"的最终计算，与场馆容量、票房热度等因素叠加。

#### Step 4：差异化校验清单

输出以下三项清单，供 Layer 5 设计SKU时强制参照：
1. **需规避的品类**：与艺人现有长期产品重复的品类
2. **需规避的视觉符号**：艺人个人IP已大量铺货的符号/形象
3. **必须强化的"限定专属"元素**：场馆名、日期、编号、限定色等不可复制要素

#### Step 5（可选）：渠道协同建议

若艺人自有渠道（尤其直播带货）较强势，评估是否适用以下协同方案：
- 方案A：限定编号款 + 直播间专属补货
- 方案B：艺人长期爆款的"演唱会特别版"
- 方案C：与艺人自有品牌的一次性联名

标注"需与艺人团队对接确认"。

### Layer 1.5 输出小结格式

```
## 艺人自有IP生态与分流率分析报告

### 渠道盘点
```

## Layer 2｜演唱会情绪系统与篇章结构分析

### 目标
将演唱会解码为**情绪产品的设计蓝图**，并识别演唱会是否存在"篇章叙事结构"以驱动系列化产品设计。

### 执行步骤

#### A. 主题解析
搜索演唱会官宣信息、主视觉海报、官方宣传文案。

提取：
- 主主题（科幻/怀旧/情绪/嘉年华/史诗等）
- 情绪关键词（3-5个）
- 世界观设定
- 叙事方向

#### B. 视觉元素提取
从官宣物料中提取：
- 主色调（精确到色值范围）
- 辅色
- 主视觉元素（图形/符号/材质感）
- 字体风格
- 整体调性（冷/暖/中性）

#### C. 篇章/章节结构识别（新增，高价值）

> 许多大型巡演会设定分篇章叙事结构（如：第一幕「少年」→ 第二幕「流浪」→ 第三幕「归来」），这是驱动"系列化产品"设计的天然逻辑。

**检索方式**：搜索 `艺人名 演唱会 篇章`、`艺人名 巡演 章节`、`艺人名 演唱会 setlist 分段`

**判断与处理**：

| 情况 | 处理方式 |
|------|---------|
| 有明确篇章划分（官方有命名） | 提取每个篇章的意象/情绪/视觉关键词，作为分层系列产品的独立设计方向 |
| 有隐性段落感（高低潮有规律但无官方命名） | 自行归纳2-3个"情绪段落"，用于绑定不同SKU的情感定位 |
| 无明显结构 | 按标准情绪曲线处理即可 |

**篇章产品化逻辑**（适用于有篇章结构的演唱会）：

- 每个篇章对应一组限定SKU或一款限定色/限定图案
- 系列产品可形成"集齐感"，激励粉丝多件购买
- 礼盒可按篇章命名和设计（如：「第一章·少年礼盒」）
- 引流款可做通用款，品牌款/稀缺款可做篇章专属款

#### D. 情绪曲线构建

根据艺人演唱会历史规律与本次主题，构建情绪时间轴：

| 阶段 | 时长估算 | 情绪状态 | 代表曲目（如有） | 适配SKU方向 |
|------|---------|---------|----------------|------------|
| 开场 | | 兴奋/期待 | | 应援类 |
| 上升段 | | 热烈/投入 | | 互动类 |
| 抒情段 | | 共情/泪点 | | 歌词/情感类 |
| 高潮段 | | 释放/狂欢 | | 记录类 |
| Ending | | 不舍/纪念 | | 纪念/收藏类 |

### Layer 2 输出小结格式

```
## 演唱会情绪地图与篇章结构

**主题**：[主题名称]
**核心情绪**：[情绪1] + [情绪2] + [情绪3]
**主色调**：[颜色描述，附HEX参考]
**视觉核心元素**：[元素列表]

**篇章结构**：[有/无/隐性]
[如有]：
- 篇章1「[名称]」：意象[X] / 情绪[X] / 适配SKU方向[X]
- 篇章2「[名称]」：意象[X] / 情绪[X] / 适配SKU方向[X]
- ...
[系列产品化建议]：[描述集齐逻辑与系列命名方向]

**情绪曲线**：
开场[兴奋] → 中段[沉浸] → 抒情[共情] → 高潮[释放] → 结尾[不舍]

**情绪×SKU绑定逻辑**：
- [情绪/篇章] → 推荐 [SKU类型]
- [情绪/篇章] → 推荐 [SKU类型]
```

## Layer 5｜商品决策与爆款预测

> 综合 Layer 1-4（特别是 Layer 1.5 的分流率数据）所有分析结果，生成SKU矩阵

---

### ⛔ 首要警告：禁止从品类库出发选品

**造成"所有艺人产品都很像"的根本原因**：先想"我们要做T恤、钥匙扣、金属徽章"，然后再往里套艺人元素。这是错误的路径。

**正确路径是反向的**——先从 Layer 1 的「艺人专属符号词库」出发，问：

> **"这个符号/情绪/意象，最适合被做成什么形态的产品？"**

然后再把答案套进四象限定价结构。

看数据中的高差异化案例：
- 「五球」符号 → 拼图冰箱贴 + 胡萝卜手机支架（不是先想做冰箱贴，而是"五球能做什么"）
- 「快乐」情绪 → 发声趣味罐 + NFC感应贴（触碰就播放，快乐立刻发生）
- 「磁带时代感」符号 → 磁带造型钥匙扣（复古符号直接物化）
- 「龙拳」曲目IP → 大娃造型荧光棒（具体IP形象做成应援工具）
- 「八花」演唱会专属视觉 → 毛绒挂件（演唱会视觉符号直接产品化）

---

### Step 0：专属符号产品化推导（必须最先执行）

**在生成任何SKU之前**，对 Layer 1 词库中的每一个「标志性符号/独有元素」，逐一完成以下推导：

| 符号/元素 | 符号特征 | 产品化方向候选 | 可行性判断 |
|---------|---------|--------------|----------|
| [符号1] | 可视化图形/文字/声音/互动 | [候选品类A] / [候选品类B] | 选[A]，原因：[原因] |
| [符号2] | | | |
| [演唱会专属意象] | | | |
| [粉丝圈专属文化符号] | | | |

**推导规则**：
- 图形类符号（五球、八花、龙拳角色）→ 优先考虑立体化/毛绒化/异形化，而非平面印刷
- 声音/音乐类符号（声纹、代表曲、口头禅）→ 优先考虑发声产品/声纹暗纹/歌词载体
- 互动/情绪类符号（「快乐」「陪伴」「干杯」）→ 优先考虑有互动体验的产品形态（NFC、发光、可触发）
- 时代感/怀旧类符号（磁带、黑胶）→ 优先考虑同类型产品的造型化演绎
- 稀缺/仪式感类符号（编号、纪念日）→ 优先考虑带唯一标识的收藏品形态

**本步骤输出**：「符号驱动SKU候选清单」——这是后续产品矩阵的**第一优先来源**。

### ⚠️ 同质化品类预警：使用前必须自问

以下品类在实际鸟巢演唱会周边中，**几乎所有艺人都有**（出现率83-100%）：

| 品类 | 出现率 | 正确使用姿势 | 错误使用方式 |
|------|------|------------|------------|
| 应援棒/LED棒 | 100% | 外形/互动机制做出专属差异（大娃造型/场控特效）| 通用造型换色+印logo |
| 短袖T恤 | 100% | 图案必须来自符号词库，不能是通用印花 | 加个艺人名字/logo的普通T恤 |
| 亚克力小件（钥匙扣/卡套） | 83% | 异形切割+专属符号造型 | 方形/圆形通用亚克力 |
| 金属徽章套组 | 67% | 必须有叙事逻辑（篇章/情绪系列），不能是单纯头像 | 随机图案凑成套 |

> 若 Step 0 符号推导后没有产生任何新品类候选，且最终方案里以上四类全部出现，请重新执行 Step 0。

### ⚠️ 分流率差异化校验（执行前必读）

生成每一个SKU时，还须对照 Layer 1.5 输出的"差异化要求清单"逐一检查：

1. 该SKU品类是否在"需规避的品类"列表中？若是，需重新设计或调整工艺/版型/定位使其明显区分
2. 该SKU视觉符号是否在"需规避的视觉符号"列表中？若是，需替换为本场演唱会专属符号
3. 该SKU是否包含至少一项"必须强化的限定专属元素"（编号/日期/场馆元素等）？

任何SKU若前两项判定"高重复"且第三项"无"，标记为 **🔴 高分流风险SKU**，建议替换。

### 爆款概率评分模型

每个SKU按以下7个维度评分（各1-5分），总分35分：

| 维度 | 权重系数 | 说明 |
|------|---------|------|
| 情绪强度 | ×2 | 与粉丝当前情绪热点契合度 |
| 社交传播性 | ×2 | 适合拍照/分享/炫耀的程度 |
| 粉丝契合度 | ×2 | 与粉丝审美偏好的匹配度 |
| 收藏价值 | ×1 | 未来是否会升值/保留 |
| 使用频率 | ×1 | 日常使用频率 |
| 城市适配度 | ×1 | 对本城市消费者的吸引力 |
| 潮流适配度 | ×1 | 与当前文创趋势的契合度 |

**爆款概率等级**：
- 总分 ≥ 28分 → ⭐⭐⭐ 强烈推荐主打款
- 总分 22-27分 → ⭐⭐ 稳健款
- 总分 15-21分 → ⭐ 补充款
- 总分 < 15分 → 建议放弃或重新设计

> 💡 注意：爆款概率评分反映的是"产品本身的吸引力"，而分流率调整系数反映的是"市场需求被分散的程度"。两者独立计算，最终首批数量 = 基础估算 × 分流率系数，但爆款评分不直接乘入分流率系数。

### 高级稀缺性工艺体系（核心竞争壁垒）

> 行业90%供应商只做印花/印LOGO/印歌词，以下是真正让方案通过经纪公司筛选的差异化能力。
> 详细工艺选型参考见 `references/sku-library.md`。

#### 工具1：数据化专属暗纹设计

将本场演唱会的**独有数据**转化为产品的视觉底纹/肌理，做到"每一件都属于这一场"。

| 数据来源 | 转化方式 | 适用产品 |
|---------|---------|---------|
| 本场演唱会代表曲目的声纹波形 | 提取波形线条，转化为产品底纹/压纹/包装暗纹 | T恤内里、礼盒内衬、金属章纹路 |
| 演唱会场馆灯光节拍数据/高光时刻 | 转化为抽象光轨图案 | 印花图案、透卡底纹 |
| 演出曲目列表/歌词字频 | 微缩字体排版为底纹 | 包装内页、票根套内层 |
| 演唱会举办时间（年/月/日/时/分） | 转化为极简坐标系/数字排列 | 金属刻印、压纹 |

**操作方法**：向设计团队提供"数据源描述"，指定转化形式，输出为矢量图案。

#### 工具2：隐藏彩蛋设计（肉眼低调，有条件可见）

> 这是粉丝"发现-尖叫-分享"链路的触发器，社交传播力极强。

| 彩蛋类型 | 实现方式 | 适用场景 |
|---------|---------|---------|
| 灯光彩蛋 | 用荧光/夜光油墨印刷隐藏图案，演唱会灯光下才可见 | T恤、毛巾、帆布袋 |
| 扫码彩蛋 | 产品上印隐藏二维码，扫后跳转艺人限定内容（视频/音频/壁纸） | 礼盒内卡、贴纸 |
| 局部放大彩蛋 | 宏观看是正常图案，放大后出现隐藏歌词/符号 | 海报、艺术印刷品 |
| 角度彩蛋 | 不同角度下呈现不同图案（幻彩/镭射材质） | 亚克力卡、金属章 |

> ⚠️ 彩蛋设计必须基于 Layer 1 的「艺人专属符号词库」，彩蛋内容必须是"懂的人才会尖叫"的艺人专属元素，而非通用装饰。

#### 工具3：唯一编号体系

将"稀缺性"从感觉变为可验证的事实。

| 编号类型 | 格式示例 | 适用SKU | 成本影响 |
|---------|---------|---------|---------|
| 全球唯一流水编号 | No.0001/2000 | 稀缺款礼盒、限量海报 | 低（印刷/激光刻印） |
| 场次专属编码 | BJ-01-0234（北京场第1场第234号） | 多城巡演限定款 | 低 |
| 双重验证编码 | 产品编号 + 证书编号对应 | 高端收藏款 | 低-中 |
| 区块链数字凭证 | 实体产品绑定链上NFT | 极高端收藏款 | 高，需评估 |

> 编号体系必须在策划书中明确说明"限量总数"和"编号规则"，限量数量是对外宣传的核心卖点。

#### 工具4：差异化工艺选型

根据艺人调性和SKU档位，匹配相应工艺：

| 工艺 | 调性适配 | 成本参考 | 视觉效果 |
|------|---------|---------|---------|
| 浮雕/凹版压印 | 高级感、沉稳、收藏调性 | 中 | 触觉体验强，无色 |
| 磨砂哑膜 | 极简、成熟、低调 | 低 | 减少反光，手感柔 |
| 细闪/金属感材质 | 华丽、舞台感、仪式感 | 中 | 光线下闪烁 |
| 金属拉丝 | 工业感、力量感、男性化艺人 | 中-高 | 金属质感强 |
| 局部烫金/烫银 | 奢华、仪式感、收藏属性 | 中 | 点睛效果 |
| 幻彩/镭射 | 年轻、潮流、视觉冲击 | 低-中 | 强烈光变效果 |
| 双层夹层结构 | 隐藏彩蛋载体、解密感 | 中-高 | 独特开合体验 |
| 磁吸结构 | 高级礼盒、科技感 | 中-高 | 开合体验高级 |

> 每个 Layer 5 的SKU在设计方向字段中，应明确标注推荐工艺选型，并说明与艺人调性的匹配逻辑。

## 最终汇总输出指引

完成所有7层分析后，输出《演唱会文创全案策划书》，格式要求：

1. **封面页**：艺人名 × 演唱会名 × 城市 × 日期 + CMIS v5.1 出品
2. **执行摘要**：核心洞察3条 + 分流率等级与产量调整系数 + 推荐主打SKU3个 + 高级稀缺性核心方案1句话 + 预期收入区间
3. **各章节按顺序排列**（参照SKILL.md中的14章节表）
4. **附录A**：艺人专属元素词库 + 禁用黑名单（完整版，供设计团队直接使用）
5. **附录B**：高级稀缺性工艺方案详细说明（声纹暗纹数据源、彩蛋实现说明、编号规则）
6. **附录C**：每个 SKU 的「单品设计执行包」合集（PART 1 中文 brief + PART 2 正向 Prompt + PART 3 反向 Prompt）

> ⚠️ 附录C是策划书中体量最大、最不能缩水的部分。策划与最终设计之间的偏差，90%来自这里信息量不足。每个SKU的执行包不能合并、不能省字段、不能用关键词列表代替完整描述。

整份策划书应足够专业，可直接交给经纪公司、设计团队（含 AI 生图操作员）、生产团队与销售团队使用。
