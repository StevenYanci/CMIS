# CMIS — 6层分析执行规范

## Layer 1｜数据采集与艺人分析

### 目标
通过 web search 构建艺人的**实时全景画像**，重点关注"当下情绪"而非历史标签。

### 必须检索的内容

#### A. 基础资料
- 艺人/组合简介、出道背景
- 代表作品（歌曲/专辑/影视）
- 历史演唱会规模与口碑
- 官方应援色、粉丝昵称
- 粉丝文化特征（应援方式、圈内文化）

#### B. 实时热点（最重要）
搜索关键词参考：`艺人名 最近 热搜`、`艺人名 小红书 2024`、`艺人名 粉丝`

重点提取：
| 项目 | 内容 |
|------|------|
| 最近热搜话题 | |
| 最近爆火内容 | |
| 粉丝当前情绪方向 | |
| 社交高传播内容 | |
| 是否有争议或特殊事件 | |

> ⚠️ 如果艺人最近因"陪伴感"爆火，SKU应强化情绪陪伴方向；如果因"励志"走红，则主打成就与见证感。热点直接决定方向。

#### C. 粉丝审美分析
判断粉丝群体的主流审美偏向（可多选，需标注权重）：

| 审美类型 | 权重 | 代表符号 |
|---------|------|---------|
| 高级极简 | | 黑白灰、无衬线字体 |
| 可爱风 | | 圆润、马卡龙色 |
| 潮流风 | | 街头、解构、logo |
| 文艺风 | | 胶片感、文字、复古 |
| 二次元风 | | 动漫美学、夸张色彩 |
| 复古风 | | 旧纸质、黑胶、磁带 |
| 朋克/暗黑风 | | 金属、对比色 |

> ⚠️ 审美判断错误是文创失败的第一大原因，需综合多个数据源交叉验证。

**当前情绪热点**：
- 热点1：[内容] → 对文创的影响方向
- 热点2：[内容] → 对文创的影响方向

**粉丝主流审美**：[类型] (权重[%]) + [类型] (权重[%])

## Layer 2｜演唱会情绪系统分析

### 目标
将演唱会解码为**情绪产品的设计蓝图**。

### 执行步骤

#### A. 主题解析
搜索演唱会官宣信息、主视觉海报、官方宣传文案。

提取：
- 主主题（科幻/怀旧/情绪/嘉年华/史诗等）
- 情绪关键词（3-5个）
- 世界观设定
- 叙事方向

#### B. 视觉元素提取
从官宣物料中提取：
- 主色调（精确到色值范围）
- 辅色
- 主视觉元素（图形/符号/材质感）
- 字体风格
- 整体调性（冷/暖/中性）

#### C. 情绪曲线构建

根据艺人演唱会历史规律与本次主题，构建情绪时间轴：

| 阶段 | 时长估算 | 情绪状态 | 代表曲目（如有） | 适配SKU方向 |
|------|---------|---------|----------------|------------|
| 开场 | | 兴奋/期待 | | 应援类 |
| 上升段 | | 热烈/投入 | | 互动类 |
| 抒情段 | | 共情/泪点 | | 歌词/情感类 |
| 高潮段 | | 释放/狂欢 | | 记录类 |
| Ending | | 不舍/纪念 | | 纪念/收藏类 |

### Layer 2 输出小结格式

```
## 演唱会情绪地图

**主题**：[主题名称]
**核心情绪**：[情绪1] + [情绪2] + [情绪3]
**主色调**：[颜色描述，附HEX参考]
**视觉核心元素**：[元素列表]

**情绪曲线**：
开场[兴奋] → 中段[沉浸] → 抒情[共情] → 高潮[释放] → 结尾[不舍]

**情绪×SKU绑定逻辑**：
- [情绪] → 推荐 [SKU类型]
- [情绪] → 推荐 [SKU类型]
```

#### 联名权重判定

根据两项评分，输出联名类型：

| 类型 | 判定条件 | 场馆元素策略 |
|------|---------|------------|
| **艺人主导型** | 艺人≥8分，场馆≤5分 | 场馆作为隐性细节（水洗标/经纬度/压纹） |
| **平衡联名型** | 两者差值≤2分，均≥6分 | 双IP共同表达，场馆可上主视觉 |
| **场馆借势型** | 场馆≥8分，艺人≤6分 | 场馆元素可作为核心卖点 |

| 层级 | 表现方式 | 适用场景 |
|------|---------|---------|
| **隐性层**（推荐超级艺人） | 水洗标、经纬度、日期压纹、内页暗纹 | 艺人主导型 |
| **辅助层** | 场馆线稿、建筑剪影、包装封套 | 平衡联名型 |
| **主视觉层** | 场馆建筑主图、城市主视觉 | 场馆借势型/城市限定款 |

## Layer 5｜商品决策与爆款预测

> 综合前4层所有分析结果，生成SKU矩阵

### SKU四象限结构

每个象限的SKU数量建议：

| 象限 | 定位 | 数量 | 价格范围 |
|------|------|------|---------|
| **引流款** | 低价高传播，冲动消费 | 2-3个 | ¥15-49 |
| **利润款** | 高频使用，情绪价值强 | 3-4个 | ¥69-199 |
| **品牌款** | 提升调性，社交传播 | 1-2个 | ¥199-499 |
| **稀缺款** | 高收藏价值，限量 | 1-2个 | ¥299-999+ |

### SKU表格模板

每个SKU必须包含：

| 字段 | 内容 |
|------|------|
| SKU名称 | |
| 所属象限 | 引流/利润/品牌/稀缺 |
| 产品描述 | 2-3句话 |
| 情绪绑定 | 对应演唱会哪个情绪段 |
| 核心卖点 | 3个 |
| 设计方向 | 简述视觉概念 |
| 场馆元素层级 | 隐性/辅助/主视觉/无 |
| 建议零售价 | |
| 建议首批数量 | |
| 成本参考 | |
| 毛利率估算 | |

### 爆款概率评分模型

每个SKU按以下7个维度评分（各1-5分），总分35分：

| 维度 | 权重系数 | 说明 |
|------|---------|------|
| 情绪强度 | ×2 | 与粉丝当前情绪热点契合度 |
| 社交传播性 | ×2 | 适合拍照/分享/炫耀的程度 |
| 粉丝契合度 | ×2 | 与粉丝审美偏好的匹配度 |
| 收藏价值 | ×1 | 未来是否会升值/保留 |
| 使用频率 | ×1 | 日常使用频率 |
| 城市适配度 | ×1 | 对本城市消费者的吸引力 |
| 潮流适配度 | ×1 | 与当前文创趋势的契合度 |

**爆款概率等级**：
- 总分 ≥ 28分 → ⭐⭐⭐ 强烈推荐主打款
- 总分 22-27分 → ⭐⭐ 稳健款
- 总分 15-21分 → ⭐ 补充款
- 总分 < 15分 → 建议放弃或重新设计

### Layer 5 输出格式

输出完整SKU矩阵表格 + 爆款概率排行榜，格式：

```
## SKU战略矩阵

[完整SKU表格]

## 爆款概率排行榜

第1名：[SKU名称] | 总分[XX]分 | ⭐⭐⭐ 强烈推荐
第2名：...

## 定价策略总结
- 引流款入场价：¥[XX]
- 主力消费带：¥[XX]-¥[XX]
- 高端收藏价：¥[XX]+
- 建议套装组合：[描述]
```

## 最终汇总输出指引

完成所有6层分析后，输出《演唱会文创全案策划书》，格式要求：

1. **封面页**：艺人名 × 演唱会名 × 城市 × 日期 + CMIS出品
2. **执行摘要**：核心洞察3条 + 推荐主打SKU3个 + 预期收入区间
3. **各章节按顺序排列**（参照SKILL.md中的章节表）
4. **附录**：设计简报（关键词+禁忌）+ AI生图提示词合集

整份策划书应足够专业，可直接交给经纪公司、设计团队、生产团队与销售团队使用。
