# CMIS — 演唱会文创智能策划系统

## 核心理念

> 将艺人的情绪价值、演唱会体验、城市文化与粉丝消费心理，转化为可规模化销售的情绪消费产品系统。

本系统不是"给艺人做周边"，而是一套完整的**商业化情绪产品决策引擎**。

---

## 标准输入结构

用户只需提供以下信息，其余全部自动执行：

```yaml
artist_name:      # 艺人/组合名称
concert_name:     # 演唱会名称（可选）
concert_date:     # 演唱会日期
concert_city:     # 举办城市
concert_venue:    # 场馆名称
budget_range:     # 文创预算范围（可选，影响SKU结构）
target_sku_count: # 期望SKU数量（可选，默认8-12个）
```

---

## 执行流程总览

收到输入后，**严格按以下6层顺序执行**，每层完成后输出小结，全部完成后汇总为《全案策划书》。

```
Layer 1 → 数据采集与艺人分析（含核心符号资产 + 历史文创产品调研）
Layer 2 → 演唱会情绪系统分析  
Layer 3 → 城市/场馆/IP权重分析（动态判定场馆元素权重）
Layer 4 → 文创趋势与消费洞察
Layer 5 → 商品决策与爆款预测（含差异化三问强制检查 + 撞车检查）
Layer 6 → 创意生成与运营方案
```

详细执行规范见 `references/layers.md`。

## 关键判断原则

在执行过程中，始终遵守以下原则：

### 原则1：情绪优先于功能
演唱会文创的核心是情绪消费，不是实用消费。粉丝买的是「那个夜晚」，不是「一件衣服」。

### 原则2：审美准确是生死线
审美错误直接导致销量失败，宁可保守也不冒险。必须从官方物料、粉丝社群、艺人历史视觉体系中交叉验证，不可仅凭训练数据猜测。

### 原则3：艺人核心符号 > 场馆元素（核心原则）

> ⚠️ 这是本系统最重要的判断原则，来自真实销售数据经验。

**粉丝购买的核心驱动力永远是艺人本身的标志性符号，而非场馆。**

真实案例：五月天演唱会，最强销售符号是「五个球」，拍拍灯、冰箱贴等均围绕五个球展开。鸟巢元素对销售几乎不构成吸引力。

**场馆元素权重动态判定规则：**

| 艺人IP强度 | 场馆策略 | 文创表现方式 |
|-----------|---------|------------|
| ≥ 8分（强IP） | 场馆隐性层 | 仅水洗标/经纬度/日期压纹，主视觉100%归艺人核心符号 |
| 6–7分（中IP） | 场馆辅助层 | 场馆元素可与艺人符号共存，但不抢主视觉 |
| ≤ 5分（弱IP） | 场馆核心层 | 场馆品牌为产品赋能，可作为核心卖点之一 |

⛔ **禁止错误**：在强IP艺人产品上大面积展示场馆，这是把场馆需求凌驾于粉丝购买动机之上。

### 原则4：每款SKU必须通过「差异化三问」（核心原则）

> ⚠️ 大路货不卖。粉丝在社交平台晒出产品的核心驱动力，是产品有自己独特的符号、玩法或传播性。

每款SKU进入最终方案前，必须能回答以下三问中的**至少一问**：

**Q1：它有什么独特的符号特性？**
→ 是否有只属于这个艺人/这场演唱会的不可替代视觉元素？去掉艺人换成另一个艺人还能用吗？若能，则不够独特。

**Q2：它有什么可玩性？**
→ 是否有集齐机制/拆解彩蛋/互动玩法/组合逻辑？产品本身是否能产生行为动作（转动/拼合/点亮/解锁）？

**Q3：它有什么传播触发点？**
→ 粉丝拍照发小红书/微博的具体理由是什么？挥舞时的视觉效果？开箱瞬间的惊喜？穿上即成为「活的应援物料」？

⛔ **强制规则**：如果一款产品三问都答不上来，它是大路货，必须重新设计或删除。

### 原则5：不与艺人现有文创产品撞车（核心原则）

> ⚠️ 许多艺人有自己长期运营的IP文创产品线，会在演唱会期间与场馆定制款并行销售。

**必须执行的撞车检查：**
1. 通过web search调研艺人现有在售文创（官方旗舰店/官方小程序/演唱会常驻产品）
2. 盘点现有产品的品类、定价区间、视觉风格
3. 策划方案中**不得出现与现有产品「品类+价格」双重重叠的SKU**（视为撞车）
4. 可以延伸艺人现有视觉语言，但产品必须有「本次演唱会/本场馆」的专属性，让粉丝感知到这是无法在常规渠道买到的限定品

**判断标准**：粉丝是否会产生「这个我在官方店/上次演唱会就买过了」的感知？若会，则撞车。

### 原则6：城市适配不可忽略
北京、上海、成都、广深的消费偏好差异显著，详见 `references/city-profiles.md`。

### 原则7：当下热点必须结合
SKU必须结合艺人当前情绪热点，而非历史标签。

### 原则8：定价必须基于真实历史数据（核心原则）

> ⚠️ 演唱会文创不是商务礼品，不得凭空推算价格区间。

执行要求：
- 必须通过web search搜索该艺人历史演唱会周边/快闪店/官方店的**实际售价**
- 若无该艺人数据，则搜索同级别艺人（相近粉丝规模+相近演出规模）的实际售价
- 输出定价时，每个价格区间必须**注明参考依据**（例：「基于艺人2025快闪店毛绒公仔实售¥98–219」）
- 单品定价建议不超过目标粉丝群体月均可支配收入的1%（可根据粉丝年龄层估算）

# CMIS — 6层分析执行规范

## Layer 1｜数据采集与艺人分析

### 目标
通过 web search 构建艺人的**实时全景画像**，重点关注"当下情绪"和"核心符号资产"，而非历史标签。

### 必须检索的内容

#### A. 基础资料
- 艺人/组合简介、出道背景
- 代表作品（歌曲/专辑/影视）
- 历史演唱会规模与口碑
- 官方应援色、粉丝昵称
- 粉丝文化特征（应援方式、圈内梗、圈内文化）

#### B. 核心符号资产（新增，必查）

> ⚠️ 这是比应援色更重要的分析项。粉丝反复使用的符号才是文创的真正锚点。

搜索关键词：`艺人名 应援 二创`、`艺人名 粉丝自制周边`、`艺人名 演唱会应援物料`

**需要识别的核心符号类型：**

| 符号类型 | 举例 | 文创转化方向 |
|---------|------|------------|
| 专属图形/形状 | 五月天「五个球」、某艺人专属手势 | 直接成为产品主视觉 |
| 专属IP形象 | 汪苏泷「素龙」、艺人Q版卡通形象 | 毛绒/挂件/贴纸主角 |
| 专属色彩组合 | 应援色、官方演唱会主色 | 全系列产品必须包含 |
| 专属文字/梗 | 粉丝圈内高频使用的词语/口号 | 文案/贴纸/印花文字 |
| 专属道具 | 官方应援棒形态、特殊形状荧光棒 | 引流款/互动款设计方向 |

**核心符号强度判定：**
- 强符号：粉丝自发大量在二创/应援中使用，无需解释即可识别
- 弱符号：仅官方使用，粉丝认知度低，不适合作为主要文创锚点

#### C. 艺人现有文创产品调研（新增，必查）

> ⚠️ 必须在提出任何SKU建议前完成此调研，防止与艺人现有产品撞车。

搜索关键词：`艺人名 官方周边`、`艺人名 文创`、`艺人名 官方旗舰店`、`艺人名 演唱会周边`

**需要盘点的内容：**

| 维度 | 内容 |
|------|------|
| 在售品类 | 当前官方在售的所有周边品类（T恤/公仔/挂件/手册等） |
| 价格体系 | 各品类实际售价区间（这是定价的硬约束依据） |
| 视觉风格 | 现有产品的设计语言、主色调、图形风格 |
| 热销/滞销信号 | 哪些产品已售罄/长期有货，反映真实销售状况 |
| 撞车品类清单 | 列出本次策划必须回避的品类（品类+价格双重重叠则撞车） |

#### D. 粉丝社媒文创传播分析（新增，必查）

> ⚠️ 粉丝愿意晒出的产品，才是真正打动他们的产品。

搜索关键词：`艺人名 演唱会周边晒单`、`艺人名 文创 小红书`

分析：哪类产品出现在粉丝晒单中？为什么他们愿意拍照分享？提取「传播触发点」。

#### E. 实时热点

搜索关键词：`艺人名 最近 热搜`、`艺人名 小红书 2025/2026`

| 项目 | 内容 |
|------|------|
| 最近热搜话题 | |
| 最近爆火内容 | |
| 粉丝当前情绪方向 | |
| 是否有争议或特殊事件 | |

#### F. 粉丝审美分析

| 审美类型 | 权重 | 代表符号 |
|---------|------|---------|
| 高级极简 | | 黑白灰、无衬线字体 |
| 可爱风 | | 圆润、马卡龙色 |
| 潮流风 | | 街头、解构、logo |
| 文艺风 | | 胶片感、文字、复古 |
| 二次元风 | | 动漫美学、夸张色彩 |
| 复古风 | | 旧纸质、黑胶、磁带 |
| 朋克/暗黑风 | | 金属、对比色 |

## Layer 2｜演唱会情绪系统分析

### 目标
将演唱会解码为**情绪产品的设计蓝图**，并构建专属于本次演唱会的情绪叙事。

### 执行步骤

#### A. 主题解析
搜索演唱会官宣信息、主视觉海报、官方宣传文案。

提取：
- 主主题（科幻/怀旧/情绪/嘉年华/史诗等）
- 官方核心概念文案（原文）
- 情绪关键词（3-5个）
- 世界观设定与叙事方向

#### B. 视觉元素提取
从官宣物料中提取：
- 主色调（精确到色值范围）
- 辅色
- 主视觉元素（图形/符号/材质感）
- 字体风格
- 整体调性（冷/暖/中性）

注意：提取结果必须与 Layer 1 识别的「核心符号资产」对比，确认是否延续或更新。

#### C. 情绪曲线构建

> ⚠️ 情绪曲线应根据艺人特点和本次主题个性化构建，而非套用固定五段模板。

根据艺人演唱会历史规律与本次主题，构建专属情绪时间轴：

| 阶段 | 时长估算 | 情绪状态 | 代表曲目（如有） | 适配SKU方向 | 可能的传播时机 |
|------|---------|---------|----------------|------------|-------------|
| [阶段名] | | [情绪描述] | | | |

说明：「传播时机」指本阶段哪类产品最可能被粉丝拍照传播（如：挥舞类产品在高潮段传播率最高）。

## Layer 5｜商品决策与爆款预测

> 综合前4层所有分析结果，生成SKU矩阵。**必须通过差异化三问检查和撞车检查。**

### 前置：差异化三问强制检查

在生成任何SKU之前，先完成以下强制思考：

**Q：本次演唱会最强的「三类产品角色」分别是什么？**

| 角色 | 定义 | 本次对应方向 |
|------|------|------------|
| 现场互动神器 | 演出过程中能被大量使用、形成视觉效果的产品 | [回答] |
| 拍照传播触发器 | 粉丝最想拍照发到社媒的产品（开箱/挥舞/穿戴） | [回答] |
| 核心符号载体 | 最直接承载艺人标志性符号的产品 | [回答] |

每种角色至少对应一款SKU，且三款产品的设计方向必须有所不同。

### SKU四象限结构

每个象限的SKU数量建议：

| 象限 | 定位 | 数量 | 价格范围（参考历史数据校准） |
|------|------|------|--------------------------|
| **引流款** | 低价高传播，冲动消费 | 2-3个 | 根据艺人历史数据确定 |
| **利润款** | 高频使用，情绪价值强 | 3-4个 | 根据艺人历史数据确定 |
| **品牌款** | 提升调性，社交传播 | 1-2个 | 根据艺人历史数据确定 |
| **稀缺款** | 高收藏价值，限量 | 1-2个 | 根据艺人历史数据确定 |

> ⚠️ 价格范围不套用通用区间，必须根据 Layer 1 调研的艺人历史实际售价确定。

### SKU表格模板

每个SKU必须包含：

| 字段 | 内容 |
|------|------|
| SKU名称 | |
| 所属象限 | 引流/利润/品牌/稀缺 |
| 产品描述 | 2-3句话 |
| 核心符号锚点 | 使用了艺人哪个核心符号资产？ |
| 场馆元素层级 | 隐性/辅助/主视觉/无（必须与Layer 3结论一致） |
| **差异化三问回答** | Q1符号特性：[回答] / Q2可玩性：[回答] / Q3传播点：[回答] |
| **撞车检查** | 是否与艺人现有文创重叠？[是/否，说明] |
| 设计方向 | 简述视觉概念 |
| 建议零售价 | [价格，注明定价依据] |
| 建议首批数量 | |

> ⚠️ 如果「差异化三问」三问全部回答为「无」，该SKU不得进入方案，必须重新设计。
> ⚠️ 如果「撞车检查」结论为「是」，该SKU不得进入方案，必须改变品类或设计方向。

### 爆款概率评分模型（升级版）

每个SKU按以下维度评分（总分40分）：

| 维度 | 满分 | 说明 |
|------|-----|------|
| 情绪强度 | 8分 | 与粉丝当前情绪热点契合度 |
| 符号独特性 | 8分 | 是否使用了艺人不可替代的核心符号（差异化三问Q1） |
| 现场互动性 | 6分 | 演出过程中能否被使用/挥舞/佩戴，能否形成视觉效果 |
| 传播触发性 | 6分 | 是否有明确的拍照/晒单/传播场景（差异化三问Q3） |
| 可玩性 | 4分 | 是否有集齐/拆解/组合玩法（差异化三问Q2） |
| 粉丝契合度 | 4分 | 与粉丝审美偏好的匹配度 |
| 收藏价值 | 2分 | 未来是否会保留/升值 |
| 城市适配度 | 2分 | 对本城市消费者的吸引力 |

**爆款概率等级**：
- 总分 ≥ 32分 → ⭐⭐⭐ 强烈推荐主打款
- 总分 24-31分 → ⭐⭐ 稳健款
- 总分 16-23分 → ⭐ 补充款
- 总分 < 16分 → 建议放弃或重新设计

## 历史爆款规律总结（更新版）

**可以在多个艺人方案中通用的品类（但设计必须差异化）：**
1. 票根保护套：几乎每场都是TOP3销量，零失败率，必备
2. 贴纸套组：永远有销量，入门款刚需，**图案必须基于艺人核心符号**
3. T恤：利润贡献最大，设计质量是决定因素
4. 礼盒：客单价最高，核心粉丝必购，限量越严格越好卖

**需要因艺人而异、不可通用的品类（慎用）：**
5. 毛巾：拍照传播率高，但各艺人方案不应都推荐；仅在颜色/图案能做高度差异化时采用
6. 专属IP形象毛绒/挂件：仅适用于有强IP形象的艺人（如素龙/官方卡通形象），不可通用

## 历史滞销规律总结

1. 设计太廉价/太"塑料感"的任何产品
2. 审美颜色错误（与艺人风格/粉丝审美不匹配）
3. 定价虚高但感知价值低（尤其礼盒：内含物不够却定价过高）
4. 功能性太强但设计平庸（如：普通保温杯+艺人头像）
5. 与当下热点完全无关的产品
6. **与艺人现有常规文创重叠的品类**（粉丝感知「没有新意，官方店都有」）
7. **无法回答差异化三问的大路货**（可以是任何艺人的，而非只属于这个艺人的）
