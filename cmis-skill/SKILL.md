# CMIS — 演唱会文创智能策划系统

## 核心理念

> 将艺人的情绪价值、演唱会体验、城市文化与粉丝消费心理，转化为可规模化销售的情绪消费产品系统。

本系统不是"给艺人做周边"，而是一套完整的**商业化情绪产品决策引擎**。

---

## 标准输入结构

用户只需提供以下信息，其余全部自动执行：

```yaml
artist_name:      # 艺人/组合名称
concert_name:     # 演唱会名称（可选）
concert_date:     # 演唱会日期
concert_city:     # 举办城市
concert_venue:    # 场馆名称
budget_range:     # 文创预算范围（可选，影响SKU结构）
target_sku_count: # 期望SKU数量（可选，默认8-12个）
```

---

## 执行流程总览

收到输入后，**严格按以下7层顺序执行**，每层完成后输出小结，全部完成后汇总为《全案策划书》。

```
Layer 1   → 数据采集与艺人分析
Layer 1.5 → 艺人自有IP生态与长期销售渠道分析（分流率评估，核心新增）
Layer 2   → 演唱会情绪系统分析  
Layer 3   → 城市/场馆/IP权重分析
Layer 4   → 文创趋势与消费洞察
Layer 5   → 商品决策与爆款预测
Layer 6   → 创意生成与运营方案
```

详细执行规范见 `references/layers.md`。

## ⚠️ 核心新增原则：分流率思维（Diversion Rate）

> 这是本 Skill 区别于一般周边策划工具的关键。

**任何艺人都不是"裸IP"**。在筹备演唱会限定周边之前，必须先回答一个问题：

> 这位艺人本身有多少"长期、稳定、低门槛"的官方/个人IP消费渠道？这些渠道会分流多少粉丝的购买力和购买意愿到演唱会限定款之外？

### 典型场景对比

| 艺人画像 | 自有IP生态 | 对演唱会限定款的影响 |
|---------|-----------|---------------------|
| **谢霆锋型** | 自有餐饮品牌（锋味）+ 常规文创（T恤/方巾）+ 长期抖音直播带货 | 粉丝购买渠道高度分散且常态化，"鸟巢限定"稀缺性被严重稀释，**实际购买转化率明显偏低** |
| **五月天型** | 阿信个人IP经营成熟，五球毛绒等长期在固定渠道（自有店铺/合作伙伴）持续销售 | 经典符号已被"日常化"，演唱会限定款若与常规款雷同，会被粉丝视为"换皮重复购买"，**意愿降低** |
| **新人/纯演唱会型艺人** | 几乎无自有IP消费场景，粉丝购买行为高度集中在演唱会档期 | 分流率低，演唱会限定款是粉丝表达情感的**唯一/主要出口**，转化率天然更高 |

### 分流率对策略的影响（必须执行）

1. **差异化优先于雷同**：演唱会限定款必须与艺人现有长期产品在**品类、视觉符号、材质工艺**上明显区分，不能是"加个场馆logo的同款"
2. **量化下调生产基数**：分流率越高，演唱会限定款的预估销量基数应相应下调（具体系数见 Layer 1.5）
3. **强化"限定专属价值"**：当艺人长期渠道发达时，演唱会限定款必须更突出"此刻/此地/此场"的不可复制性（编号、日期、场馆专属元素层级可适当上调）
4. **渠道协同而非对立**：如果艺人有强势直播带货渠道，可考虑设计"演唱会限定款+主播间专属补货"的组合策略，而非视为竞争关系一味回避

完成所有分析后，输出标准格式的《演唱会文创全案策划书》，包含以下章节：

| # | 章节 | 核心内容 |
|---|------|---------|
| 1 | 艺人IP分析 | 国民度、粉丝画像、审美偏好、情绪热点 |
| 2 | 艺人自有IP生态与分流率分析 | 自有品牌/常规文创/直播带货渠道梳理，分流率评估 |
| 3 | 演唱会情绪地图 | 主题解析、情绪曲线、视觉元素提取 |
| 4 | 城市与场馆分析 | 消费能力、地域文化、场馆IP权重 |
| 5 | 文创趋势报告 | 当下爆款品类、材质风格、消费趋势 |
| 6 | SKU战略矩阵 | 引流/利润/品牌/稀缺款完整结构（含分流率调整后产量） |
| 7 | 爆款预测排行 | 每个SKU的爆款概率评分 |
| 8 | 设计关键词 | 视觉/情绪/材质/禁忌关键词 |
| 9 | 包装与文案方向 | 开箱体验、仪式感、社交传播点 |

| 12 | 风险提示 | 库存风险、设计风险、粉丝预期风险、分流风险 |

## 关键判断原则

在执行过程中，始终遵守以下原则：

1. **情绪优先于功能**：演唱会文创的核心是情绪消费，不是实用消费
2. **审美准确是生死线**：审美错误直接导致销量失败，宁可保守也不冒险
3. **场馆元素要克制**：除非场馆本身有极高IP价值，否则场馆元素应作为隐性细节而非主视觉
4. **城市适配不可忽略**：北京、上海、成都、广深的消费偏好差异显著
5. **当下热点必须结合**：SKU必须结合艺人当前情绪热点，而非历史标签
6. **分流率必须量化**：艺人自有IP生态越成熟，演唱会限定款越要做"差异化+不可复制"，且产量预估要相应保守

## Web Search 指引

执行过程中，**必须使用 web search** 获取以下实时信息：

- 艺人最近热搜、争议、爆火内容（微博/小红书/抖音）
- 演唱会官宣信息与主视觉
- 城市消费数据与文娱消费习惯
- 当前文创行业爆款趋势
- 粉丝社群讨论与审美偏好
- **艺人自有品牌/文创产品线及销售渠道**（官方店铺、直播带货、长期联名等）
- **该艺人往届演唱会限定周边的实际销售反馈**（如有公开信息）

不可仅依赖训练数据，因为**"当下情绪"和"艺人自有渠道现状"是周边成败的核心变量**。
