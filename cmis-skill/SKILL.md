# CMIS — 演唱会文创智能策划系统

## 核心理念

> 将艺人的情绪价值、演唱会体验、城市文化与粉丝消费心理，转化为可规模化销售的情绪消费产品系统。

本系统不是"给艺人做周边"，而是一套完整的**商业化情绪产品决策引擎**。

---

## 标准输入结构

用户只需提供以下信息，其余全部自动执行：

```yaml
artist_name:      # 艺人/组合名称
concert_name:     # 演唱会名称（可选）
concert_date:     # 演唱会日期
concert_city:     # 举办城市
concert_venue:    # 场馆名称
budget_range:     # 文创预算范围（可选，影响SKU结构）
target_sku_count: # 期望SKU数量（可选，默认8-12个）
```

---

## 执行流程总览

收到输入后，**严格按以下6层顺序执行**，每层完成后输出小结，全部完成后汇总为《全案策划书》。

```
Layer 1 → 数据采集与艺人分析（含核心符号资产 + 历史文创产品调研）
Layer 2 → 演唱会情绪系统分析  
Layer 3 → 城市/场馆/IP权重分析（动态判定场馆元素权重）
Layer 4 → 文创趋势与消费洞察
Layer 5 → 商品决策与爆款预测（含差异化三问强制检查 + 撞车检查）
Layer 6 → 创意生成与运营方案
```

详细执行规范见 `references/layers.md`。

## 关键判断原则

在执行过程中，始终遵守以下原则：

### 原则1：情绪优先于功能
演唱会文创的核心是情绪消费，不是实用消费。粉丝买的是「那个夜晚」，不是「一件衣服」。

### 原则2：审美准确是生死线
审美错误直接导致销量失败，宁可保守也不冒险。必须从官方物料、粉丝社群、艺人历史视觉体系中交叉验证，不可仅凭训练数据猜测。

### 原则3：艺人核心符号 > 场馆元素（核心原则）

> ⚠️ 这是本系统最重要的判断原则，来自真实销售数据经验。

**粉丝购买的核心驱动力永远是艺人本身的标志性符号，而非场馆。**

真实案例：五月天演唱会，最强销售符号是「五个球」，拍拍灯、冰箱贴等均围绕五个球展开。鸟巢元素对销售几乎不构成吸引力。

**场馆元素权重动态判定规则：**

| 艺人IP强度 | 场馆策略 | 文创表现方式 |
|-----------|---------|------------|
| ≥ 8分（强IP） | 场馆隐性层 | 仅水洗标/经纬度/日期压纹，主视觉100%归艺人核心符号 |
| 6–7分（中IP） | 场馆辅助层 | 场馆元素可与艺人符号共存，但不抢主视觉 |
| ≤ 5分（弱IP） | 场馆核心层 | 场馆品牌为产品赋能，可作为核心卖点之一 |

⛔ **禁止错误**：在强IP艺人产品上大面积展示场馆，这是把场馆需求凌驾于粉丝购买动机之上。

### 原则4：每款SKU必须通过「差异化三问」（核心原则）

> ⚠️ 大路货不卖。粉丝在社交平台晒出产品的核心驱动力，是产品有自己独特的符号、玩法或传播性。

每款SKU进入最终方案前，必须能回答以下三问中的**至少一问**：

**Q1：它有什么独特的符号特性？**
→ 是否有只属于这个艺人/这场演唱会的不可替代视觉元素？去掉艺人换成另一个艺人还能用吗？若能，则不够独特。

**Q2：它有什么可玩性？**
→ 是否有集齐机制/拆解彩蛋/互动玩法/组合逻辑？产品本身是否能产生行为动作（转动/拼合/点亮/解锁）？

**Q3：它有什么传播触发点？**
→ 粉丝拍照发小红书/微博的具体理由是什么？挥舞时的视觉效果？开箱瞬间的惊喜？穿上即成为「活的应援物料」？

⛔ **强制规则**：如果一款产品三问都答不上来，它是大路货，必须重新设计或删除。

### 原则5：不与艺人现有文创产品撞车（核心原则）

> ⚠️ 许多艺人有自己长期运营的IP文创产品线，会在演唱会期间与场馆定制款并行销售。

**必须执行的撞车检查：**
1. 通过web search调研艺人现有在售文创（官方旗舰店/官方小程序/演唱会常驻产品）
2. 盘点现有产品的品类、定价区间、视觉风格
3. 策划方案中**不得出现与现有产品「品类+价格」双重重叠的SKU**（视为撞车）
4. 可以延伸艺人现有视觉语言，但产品必须有「本次演唱会/本场馆」的专属性，让粉丝感知到这是无法在常规渠道买到的限定品

**判断标准**：粉丝是否会产生「这个我在官方店/上次演唱会就买过了」的感知？若会，则撞车。

### 原则6：城市适配不可忽略
北京、上海、成都、广深的消费偏好差异显著，详见 `references/city-profiles.md`。

### 原则7：当下热点必须结合
SKU必须结合艺人当前情绪热点，而非历史标签。

### 原则8：定价必须基于真实历史数据（核心原则）

> ⚠️ 演唱会文创不是商务礼品，不得凭空推算价格区间。

执行要求：
- 必须通过web search搜索该艺人历史演唱会周边/快闪店/官方店的**实际售价**
- 若无该艺人数据，则搜索同级别艺人（相近粉丝规模+相近演出规模）的实际售价
- 输出定价时，每个价格区间必须**注明参考依据**（例：「基于艺人2025快闪店毛绒公仔实售¥98–219」）
- 单品定价建议不超过目标粉丝群体月均可支配收入的1%（可根据粉丝年龄层估算）

## Web Search 必查清单（执行时逐项完成）

| 查询项目 | 重要性 | 影响判断 |
|---------|-------|---------|
| 演唱会官宣信息与主视觉 | ⭐⭐⭐ | Layer 2 主题解析 |
| 艺人标志性核心符号（粉丝二创/应援反复使用的视觉元素） | ⭐⭐⭐ | 原则3，Layer 5 SKU设计方向 |
| 艺人现有文创产品线（官方店品类+定价） | ⭐⭐⭐ | 原则5，撞车检查 |
| 艺人历史演唱会周边实际售价 | ⭐⭐⭐ | 原则8，定价依据 |
| 粉丝在社媒晒出的文创内容（分析传播原因） | ⭐⭐⭐ | 原则4，差异化三问 |
| 艺人最近热搜/爆火内容/粉丝情绪 | ⭐⭐⭐ | Layer 1 情绪热点 |
| 粉丝审美偏好/圈内文化/应援色 | ⭐⭐ | Layer 1 粉丝画像 |
| 当前文创行业爆款趋势 | ⭐⭐ | Layer 4 趋势报告 |
| 城市消费数据 | ⭐ | Layer 3 城市分析 |

不可仅依赖训练数据，**「当下情绪」「真实售价」「现有文创产品」是周边成败的三大核心变量**。
